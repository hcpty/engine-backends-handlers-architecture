An app architecture.

### Engine-Extension-Activity Architecture

使用Engine-Extension-Activity架构的系统包含三种元素：Engine、Extension和Activity。其中：
- Engine负责运行Activities。
- Extensions负责拓展Engine的API。
- Activities负责使用Engine和Extensions提供的API处理任务。
