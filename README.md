An app architecture.

### Engine-Extensions-Handlers Architecture

使用Engine-Extensions-Handlers架构的系统包含三种元素：Engine、Extension和Handler。其中：
- Engine负责运行Extensions和Handlers。
- Extensions负责提供自定义的API。
- Handlers负责使用Engine和Extensions提供的API处理任务。
